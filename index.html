Anki Tools

  • audio-files-check.py - Python script that checks whether audio files from audio-table-prep exist, requires replacing a hardcode multiline string for each batch
  • anki-sheet-audacity-labels-copy.ahk - AHK script for quick copying of label names into audacity
  • audio-table-prep - Converts tab separated value (TSV) word pairs into normalized audio filenames (for use as Audacity labels) and Anki [sound:] fields
  • pair-paste-tabber - Paste a line and generate versions where each space is replaced by a tab for quick copying into tab-separated fields; for cleaning up vocabulary 'tables' copied from OCR'd (text recognition/image-to-text/document scanning) study book scans/screenshots
  • (semi-related) locatools - localization tools, might be useful too
